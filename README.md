# Project-Portfolio
Curated portfolio of AI/ML, Cybersecurity, Blockchain, and Java projects.

# ACTREC Frontend (IIT Bombay)

Developed a Java-based visual programming interface during my internship at IIT Bombay that enables users to build computational workflows through a drag-and-drop block editor. The application supports type-safe block connections, custom block creation, topological execution ordering, cycle detection, canvas navigation, and automatic workflow generation via an execution engine. This project demonstrates strong object-oriented design, GUI development with Java Swing, graph algorithms, and software engineering principles.

**Repository**
[ACTREC Frontend](https://github.com/tanishasenapati13/ACTREC-frontend)

# Audio Job Interview Chatbot

Developed a voice-enabled AI chatbot that conducts interactive mock job interviews using Large Language Models (LLMs). The application leverages Whisper for speech-to-text transcription, Hugging Face models for generating context-aware interview questions and feedback, and Google Text-to-Speech (gTTS) for natural voice responses. Built with a Django frontend and Flask backend, the system provides an immersive platform for interview preparation through real-time voice conversations.

**Repository**
[Audio_chatbot](https://github.com/tanishasenapati13/Audio-chatbot)

# Intelligent Secure Transaction System (Project Contributor)

Contributed to the development of an AI-powered transaction security system that combines machine learning, cryptography, and blockchain to detect and prevent fraudulent financial transactions. My primary contributions included designing and implementing the AI/ML risk scoring components and developing the frontend interface for the system. The project integrates a PyTorch-based risk scoring model, an adaptive policy engine, ECDSA digital signatures for secure transaction validation, and a blockchain-backed audit ledger, with FastAPI providing RESTful API services.

**Repository**
[Secure_Transaction_System](https://github.com/shravya004/Secure_Transaction_System)

# Adaptive Trust-Aware Federated Intrusion Detection System (Project Contributor)

Contributed to the development of a federated intrusion detection system designed to defend against model poisoning attacks in distributed learning environments. My primary contributions included FLNET2023 dataset preprocessing, non-IID client data partitioning, and the design and implementation of a Hybrid GRU-Transformer deep learning model for network intrusion detection. The project integrates adaptive trust-aware aggregation, client reputation management, and secure federated learning to improve the robustness and accuracy of collaborative intrusion detection.

**Repository**
[FedNIDS_Member3](https://github.com/shravya004/FedNIDS)
